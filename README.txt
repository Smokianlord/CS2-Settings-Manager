CS2 Settings Manager V3

What this app does:
- Replaces both old BAT files with one cleaner Python desktop app.
- Uses a darker CS2/HUD-inspired 3D UI.
- Creates these folders beside the app/exe:
  SET MAIN
  SET TRYHARD
  SET FARMING
  SET ARMSRACE
  SET SKIN INSPECT
  SET PLAYHOUR
- Applies the selected SET folder into every Steam userdata profile.
- Lets you open the app folder, Steam userdata folder, and each SET folder.

How to run without building EXE:
1. Install Python 3 for Windows.
2. Open this folder.
3. Double-click "Run App.bat".

How to build the EXE:
1. Open this folder on Windows.
2. Double-click "Build Windows EXE.bat".
3. The final EXE will appear here:
   dist\CS2 Settings Manager.exe

Important folder rule:
Keep your SET folders beside the EXE. Example:

CS2 Tool Folder
  CS2 Settings Manager.exe
  SET MAIN
  SET TRYHARD
  SET FARMING
  SET ARMSRACE
  SET SKIN INSPECT
  SET PLAYHOUR

Steam path:
The app tries to detect:
C:\Program Files (x86)\Steam\userdata

If your Steam is installed somewhere else, use the Browse button and select your userdata folder manually.

Recommended use:
1. Click "Create SET Folders".
2. Put your config files inside the folder you want, for example SET MAIN.
3. Select that setting in the app.
4. Click "Apply Selected Settings".
